Frise de livres

Titre :

Do Androids Dream of Electric Sheep?

Auteur : Philip K. Dick

Année de parution : 1968

Genre : Science-Fiction, Fiction philosophique, Fiction noire

Description :
"Post-guerre nucléaire : les animaux, symboles de statut, deviennent des compagnons, réels ou artificiels, palliant la biodiversité perdue."

Titre :

Snow Crash

Auteur : Neal Stephenson

Année de parution : 1992

Genres : Science-Fiction, Cyberpunk, Postcyberpunk

Description :
"Un Métavers où les avatars se croisent, brouillant les limites entre réalité et simulation. "Snow Crash" évoque un virus dévastateur et une drogue virtuelle altérant la réalité."

Titre :

Altered Carbon

Auteur : Richard K. Morgan

Année de parution : 2002

Genres : Science-Fiction, Mystère

Description :
"Dans un futur où la conscience humaine se transfère d'un corps à l'autre, Takeshi Kovacs, soldat réveillé dans un nouveau corps, enquête sur le meurtre mystérieux de Laurens Bancroft pour la promesse d'immortalité."

Titre :

The Quantum Thief

Auteur : Hannu Rajaniemi

Année de parution : 2010

Genre : Science-Fiction

Description :
"Dans un futur où avancées technologiques et concepts quantiques façonnent la société, Jean le Flambeur, cambrioleur renommé libéré par la mystérieuse Mieli, devient l'épicentre de l'intrigue."

Titre :

The Windup Girl

Auteur : Paolo Bacigalupi

Année de parution : 2009

Genres : Science-Fiction, Biopunk

Description :
"Bangkok, ville résiliente face au changement climatique, attire les puissances mondiales. Les humains utilisent la biotechnologie pour créer des créatures artificielles, les "windup" ou "New People", génétiquement modifiées pour des tâches spécifiques."

Titre :

Mirrorshades : The Cyberpunk Anthology

Auteur : Bruce Sterling

Année de parution : 1986

Genre : Cyberpunk

Description :
"Nouvelles explorant des mondes futuristes cyberpunk, technologie avancée, réalité virtuelle, protagonistes marginaux confrontés à la décadence urbaine. Les récits explorent l'impact technologique, la quête d'identité et les conflits avec des entités de pouvoir dans des environnements dystopiques."
